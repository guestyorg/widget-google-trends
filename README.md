# widgetGoogleTrends
React widget for render Google Trends


## Install

```bash
npm install widget-google-trends
```

## Example

```javascript
import {WidgetGoogleTrends} from 'widget-google-trends';

<WidgetGoogleTrends word="Bitcoin" />
```
